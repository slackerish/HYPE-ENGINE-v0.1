![HYPE ENGINE LOGO](docs/img/logoexpanded.png)

Engine originally built for browser-based Friday Night Funkin' HTML5 experimentation.  
Designed to provide a clean, modular, and web-native alternative for developers.

---

## Installation:

Refer to [the Build Instructions](/docs/BUILDING.md)

---

## Customization:

Engine configuration is handled through JavaScript files and data assets.

You can enable or disable systems directly inside the engine core.

More detailed configuration documentation coming soon.

---

## Softcoding (.js/.json)

Softcoding support will allow:

- External script injection
- Modular scene loading
- Event-based logic
- Gameplay modifications without altering core files

Documentation coming soon.
